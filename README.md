Indivisible
===========
Indivisible is an optimized prime number generator written in C++.

Building
--------
This project uses CMake to build.

```bash
$ cd build/
$ cmake ..
$ make
```

To build a release build run `cmake -DCMAKE_BUILD_TYPE=Release ..` instead.
